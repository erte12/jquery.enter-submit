jQuery Enter Submit Plugin
=========

>A jQuery plugin to submit a form using enter key on the input fields.

Usage
-----------
   
#### Basic Usage
```javascript
$(".form").enter();
```

#### Using a target id
```javascript
$(".form").enter({ 
    buttonId: 'send'
});
```

#### Using a target class

```javascript
$(".form").enter({ 
    buttonClass: 'clickable'
});
```
